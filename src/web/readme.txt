			README
--------------------------------------------------------------

This is a widget to search ontologies dynamically using Ajax. The
following describes the various compoents of the widget and how to use it.

Style sheet: /cs/control.js
This contains all the styles used in the widget

Javascript:
/js/control.js contains a method to call Ajax to update the Ontology information.
In this javascript the "var url" should point to the processing script.

/js/prototype contains Ajax libraries from Prototype
http://prototype.conio.net/

Scripts
/scripts contains an exmaple of a Ajax request processing script.
It is written in CGI perl.

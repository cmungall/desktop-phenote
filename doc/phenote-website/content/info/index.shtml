Software Information

Phenote is both a standalone java application and a software toolkit.

Features of Standalone software

  • Friendly spreadsheet-like interface
  • Type-ahead-suggest (autocompletion) for terms within ontologies
  • Use of any OBO-format ontology
  • Ontology navigation and term information display
  • Bulk copy/edit/delete/sort of phenotype-genotype character entries
  • Excel-compatible output format (tab-delimited)
  • Additional export formats: phenosyntax, xml
  • configurable input fields to allow custom data entry and user-specified ontologies
  • Preconfigurations available for different communities
  • Customized configurations easily added

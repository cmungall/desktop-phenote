Phenote Software

Current Release: 1.0

Three configurations of Phenote have been created to include the ontologies most appropriate for use by those communities. The following are the webstart versions for the most current stable release:
  • Fly
  • Zebrafish
  • Human

Development Versions

Note that these versions have not been fully tested and might contain bugs. Please report any bugs you find on our tracker
  • Zebrafish v1.1

    Most notable features of this version are:

    • Ability to configure phenote input fields/ontologies for use with webstart, not just the command-line version. (For more information about how to create and use your own custom configuration, read our how-to:customize documentation.)
    • Ability to search through all loaded ontologies at once with the autocompletion feature, in addition to searching for terms in individual ontologies.

You can also checkout the latest Development version of Phenote from sourceforge.

For instructions on running Phenote, or downloading the sourcecode, visit our How-to pages.

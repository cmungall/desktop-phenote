Current Release: 1.3 [Webstart]

Newest Features:
  • Web-based documentation and Help features
  • Metadata fields added to configuration. User can name their configuration, add a description, and author
  • Dialog allows user to back out of exiting from File->Exit
  • Autocompletion search default settings configurable
  • Can post-compose with >2 terms
  • Synonyms delineated by scope in Term Info: Exact, Narrow, Broad, General, Related.
  • Obsolete replacements or consideration terms are displayed in term info with links.
  • Can configure by namespace for ontologies
  • Can automatically update ontologies during startup (configurable)
  • Added common configuration for OMIM curation for use by NCBO (ncbo.cfg)
  • Can check for updates to ontology files independently from any URL
  • Obsolete replacements or consideration terms are displayed in term info with links

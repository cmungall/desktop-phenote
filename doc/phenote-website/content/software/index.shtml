Beta Release: 1.4 beta 3 [Webstart]

Newest features:
  • feature: Native look-and-feel enabled for interface. Users should now see Mac or Windows style depending on their system
  • feature: display property values of terms in term info
  • feature: Phylogeny Chooser for NESCent/PhenoMap project. A user can import a phylogenetic tree in Newick format. The user can use the tree viewer to select rows within the character template - typically taxonomic groups.
  • feature: new field types...ID (requires ':'), read-only, ints
  • feature: config disabling of copy of field (for db ids)
  • feature: term fields can be nullified
  • feature: can specify a minimum number of chars before the autocompletion search starts to kick in. (a good number is 3 or so)
  • feature: worm group adapter makes reference terms from ref query
  • change: on term completion if adding a letter use previous completion list
  • change: build file takes parameter for webstart version deployment, and puts in jnlp
  • change: added/corrected BIRN SAO & BIRNLex ontologies
  • change: plant_grammene.cfg renamed to plant.cfg
  • change: "copy" button renamed to "duplicate" better reflecting its functionality
  • bugfix: load of file doesn't crash on post comp parsing bug
  • bugfix: doesn't load obo files specified twice, twice
  • bugfix: view menu shows up at initialization
  • bugfix: layout of term history with split pane proper

See the Release Notes for more detailed information.

Running Phenote Webstart Version(s)

Phenote is a Java application, which can be launched from the web using "Webstart". Simply click on the link below, and this will download the application to your computer and run. Future clicks on the link should recognize that you've already downloaded the application, and it will simply start the software.

Webstart: Phenote v1.1

Phenote can also be launched from our Downloads page. There you will also find any newer versions under development.

Configuring your browser for Webstart

You must have Java 1.5 or later installed on your computer. Your browser must be configured to recognize webstart appications with the 'jnlp' extension. Most Mac and Windows browsers tend to have webstart pre-installed on their browsers. Linux does not [more on running Phenote on linux...]. You must set up an association between the MIME type, which is an "application/x-java-jnlp-file".

Downloading and Installing Phenote

The most up-to-date version of phenote can be checked out with subversion using:
svn co https://phenote.svn.sourceforge.net/svnroot/obo/phenote/trunk phenote

You can also browse the subversion repository at: https://obo.svn.sourceforge.net/viewvc/obo/phenote/trunk/
Note that if you use this version, it might be buggy. Stable releases can be checked out from phenote/tags in the subversion repository.

Customizing Phenote

Phenote can be customized to allow data entry fields of your choosing, as well as the ontologies that are desired to be included for any of those fields.

At the moment it requires hand editing of a file, which is fairly painless. If you are interested in configuring your version of Phenote, please read our Instructions on how to edit the configuration file. In the future, we plan to have an easy-to-use GUI that will allow you to achieve the same results.

Note that with Phenote v1.1 or later, the user can choose a different configuration from the "File" menu. When you select "Load Configuration" from the "File" menu, you are prompted with a menu of options. Listed are several configuration styles that come standard with the Phenote software, with the current choice's radio-button selected. If you have created your own custom configuration file, it should be listed here. After selecting (the radio button) for a new configuration, press "OK", or press "Cancel" to revert to your previous selection. Any changes to your configuration settings will take effect the next time Phenote is loaded.

Making Annotations

You might find it valuable to see sample annotations. Please download this sample_data.xls file. You will need to set up your phenote to the "demo" configuration. Then choose "open" under the "file" menu. Select this file from your download location. The data should load into the spreadsheet. You can play around and manipulate the annotations.

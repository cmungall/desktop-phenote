The OBO-Edit webinar will handle live chat over IRC (Internet Relay Chat).

The best way to connect to the OBO-Edit IRC chat is to use the applet on the front page. If you decide you'd like to use your own IRC applet, connect to irc.sf.net and join the #oboedit channel.

The OBO-Edit webinar will use The Gizmo Project to handle Voice over IP communications.

The Gizmo Project is an open source program similar to Skype. Users can communicate using their computer speakers and microphone. The Gizmo Project allows several dozen users to teleconference at once, which is why this technology has been chosen.

Download and install your copy from http://www.gizmoproject.com

Once the software is installed, you will to complete some initial setup tasks, such as creating a user id and password, etc.

To join the conference call, dial 1-222-626-3348 in the phone number field near the top of the screen. If you are the first to join, you'll hear strange demo music and a message saying you are the first to call.

IMPORTANT: MUTE YOUR GIZMO PHONE IF YOU AREN'T TALKING!!!
There is a mute button near the top right corner of the Gizmo window once you are connected. The Gizmo conference call system degrades quickly as more people talk, so the only way this will work is if only one or two people have their mikes unmuted at once.

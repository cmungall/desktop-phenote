Phenote Working Group IRC Chat

When: First Tuesday of every month; 8:30am

You will need an IRC client to visit our chats. You can download any number of free clients to do this. You'll need to connect to the irc.sf.net and join the #phenote channel.

Alternatively, you can connect to the Phenote IRC chat using the applet at the bottom of this page. Simply enter a name to be used during the chat. If the name you enter is already being used by someone else, an 'anonymous' one will be assigned to you. (but you can change it once you're logged on by typing the command "/nick newname")

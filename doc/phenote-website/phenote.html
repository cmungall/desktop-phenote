Phenote


Phenote is a tool to annotate phenotypes using the Entity Quality paradigm. It takes advantage of ontologies. It uses term completion to allow curators quick access to the ontology terms they want to use.

Phenote 0.9 for fly

Phenote (standalone) 0.9 for zfin

To be clear the above webstart links for zfin & fly were made via configuration not hard wired code. Thus one could in theory configure phenote for ones own purposes.

Recent addditions include the ability to post compose terms, bul update, undo, spring framework, and synonyms now are shown in the completion list.

Phenote


Phenote is a tool to annotate phenotypes using the Entity Quality paradigm. It takes advantage of ontologies. It uses term completion to allow curators quick access to the ontology terms they want to use.

Phenote 1.4


After phenote downloads the java jar files (takes a while) it will ask you what configuration you would like to use, fly, zfin, human,worm,birn....

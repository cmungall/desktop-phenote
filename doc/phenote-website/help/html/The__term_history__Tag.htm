The <term-history> Tag

The <term-history> tag sets the default display of the term history list.

[Note:  The term history feature is still under development, so it is best to set this to false for now]

Parameters

Name Description Legal Values Required Default
enable Setting this to true sets the default Phenote to display the Term History list when first loaded.
  • true
  • false
 no false

Contents

(none)

 

Example

<ns:term-history enable="false"/>

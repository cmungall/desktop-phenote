The PhenoSyntax Adapter

Purpose:  To display ontology IDs together with commented term names.  This format can be embeded within other formats.

Extensions:  .psx .pheno

Notes:  Please visit http://www.fruitfly.org/~cjm/obd/pheno-syntax.html for more information.  The current version of phenote currently associates the genotype and phenotypes in this format, and does not follow the strict rules of pheno-syntax as described at the above link.  This will be ported to the tag-value adapter name in future versions.

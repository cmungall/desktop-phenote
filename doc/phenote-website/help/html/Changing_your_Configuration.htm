Changing your Configuration

If you would like to select a different Phenote configuration, this can be accomplished from inside of Phenote.

Select the menu item Settings->Configuration...

This will popup a menu of choices, with many configurations that come pre-packaged with phenote.  The radio button next to your currently loaded configuration will be highlighted:

To select the new configuration, click on the radio button next to the configuration of your choice, and click OK. 

You will need to quit and restart Phenote for the changes to take effect.

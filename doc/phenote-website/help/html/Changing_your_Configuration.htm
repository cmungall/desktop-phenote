Changing your Configuration

If you would like to select a different Phenote configuration, this can be accomplished from inside Phenote.

Select the menu item Settings->Set Configuration...

This will pop up a menu of choices, listing the many configurations that come pre-packaged with Phenote.  The radio button next to your currently loaded configuration will be highlighted:

To select a new configuration, click on the radio button next to the configuration of your choice, and click OK. If you decide not to change your configuration after all, you can just destroy the configuration selection window.

You will need to quit and restart Phenote for the changes to take effect.

If you would like to utilize a customized configuration, please read the section on Customizing Phenote for more information.

Changing Your Proxy Setting

Some computers use a proxy to get data over the internet. Phenote needs to download ontology files from the internet, so if your computer uses a proxy, you will need to tell Phenote that. You can change your proxy setting via the menu item Settings->Set Configuration...

Click the "Set proxy..." button at the bottom of the configuration window. A small Proxy Settings window should appear.

If you check the "Use proxy" box, you can enter a proxy name (or IP address) and port. Be sure to indicate whether your proxy uses the HTTP or SOCKS protocol.

Your proxy setting is saved in your personal configuration file (config.xml) in your Phenote configuration directory. You can turn it off or change it at any time by using the "Set Configuration" menu item (as above), and Phenote will remember your setting even after you quit and relaunch.

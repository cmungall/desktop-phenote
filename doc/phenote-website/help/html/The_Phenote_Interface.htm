The Phenote Interface

This screenshot is of Phenote when configured with the "demo" configuration and loaded with data.  It is a typical interface for phenotype annotation.

The Phenote application window is comprised of three main areas:

  • The Data Entry frame located in the upper left.
  • The Term Info frame located in the upper right.
  • The Annotation Spreadsheet located in the lower half.

The Phenote application runs best in 1024x768, but the window is easily adjusted to meet your specific size needs.

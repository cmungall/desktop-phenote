What is a Configuration?

Phenote can be customized to your needs, including the content and look&feel.  The types of things that can be configured in the configuration file include:

 

  • Fields:  free-text or ontology types
    • Display names for fields
    • Ontologi(es) for fields
    • Post-composition capability
  • Load/Save Annotation data adapters
  • Ontology repository specification
  • Enable/disable of automatic ontology updates upon startup
  • Display options:
    • Term history display

 

Currently (in version 1.2 or earlier), this configuration must be done by hand-editing an xml file

We are developing an interface inside of Phenote that will allow you to specify these options without having to fiddle with the text file. This section explains the location and process of editing a configuration file.

What is a Configuration?

Background

Phenote can be customized to your specifications, including the input fields you desire, the ontologies from which those fields are populated, and the output format used.  These items are specified in a 'configuration file' which is utilized when Phenote loads. 

Phenote comes with several configurations you can choose from.  If these do not meet your specific needs, they can be easily edited following the instructions in this help section.

Items to be specified

The configuration file allows the user to specify the following items.  Each of these is described in greater detail in the 'Customizing Phenote' help section.

  • Data adapters for loading/saving annotation data
  • Input field specification, including the ontologies used for autocompletion
  • Default settings for:
    • ontology updating during startup
    • display of the term history
    • enabling/disabling of field display

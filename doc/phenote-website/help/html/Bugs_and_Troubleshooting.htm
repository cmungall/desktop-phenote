Bugs and Troubleshooting

When installing a new version of Phenote, check the release notes for a summary of changes in the new version, any known issues, and any last-minute release notes or documentation.  These can be found at:  http://www.phenote.org/phenote-release-notes.html or in the Help->Release Notes menu within Phenote.

Bug Reports

Phenote is a work in progress. Please report any odd behavior or bugs in as much detail as possible. The Phenote Working Group is very responsive to bug reports, and your bugs will be fixed as soon as possible. To report a bug, please use our SourceForge bug tracker at:

https://sourceforge.net/tracker/?group_id=76834&atid=887913

Feature Requests

We're always happy to hear suggestions for new Phenote features. Currently Bugs and Features are reported on the same tracker.  (In the future these might be separated.)  To request a new feature, please use our Sourceforge Tracker at:

https://sourceforge.net/tracker/?group_id=76834&atid=887913

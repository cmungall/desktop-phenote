Bugs and Troubleshooting

When installing a new version of Phenote, check the README file for a summary of changes in the new version, any known issues, and any last-minute release notes or documentation.

Bug Reports

Phenote is a work in progress. Bugs can and will creep in. Please report any odd behavior or bugs immediately. The Phenote Working Group is very responsive to bug reports, and your bugs will be addressed within a few days (or sometimes hours).

To report a bug, please use our Sourceforge bug tracker at:

https://sourceforge.net/tracker/?group_id=76834&atid=887913

Feature Requests

We're always listening to suggestions for new Phenote features. Currently Bugs and Features are reported on the same tracker.  (In the future these might be separated.)  To request a new feature, please use our Sourceforge Tracker at:

https://sourceforge.net/tracker/?group_id=76834&atid=887913

Other Requests & Last Resort Bug Reporting

If you have a special problem that needs immediate attention, please email Mark Gibson (the lead Phenote developer) at mgibson@fruitfly.org or Nicole Washington at NLWashington@lbl.gov.  They will try to answer your email as soon as possible.

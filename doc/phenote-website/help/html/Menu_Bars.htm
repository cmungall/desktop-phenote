Menu Bars

Phenote contains four menus with the following functions:

  • File
    • New - clears any currently loaded annotations and lets the user start over
    • Open - prompts the user to load a set of annotations saved to their computer. 
    • Save - prompts the user to save the current annotations to a file.
    • Exit - close the Phenote Application.
  • Edit (disabled)
  • Settings
    • Search Filters - users can select the filters to be used during autocompletion on ontology fields.
      • Term Name
      • Synonyms
      • Definitions
      • Obsoletes
    • Configuration - allows the user to choose a different Phenote configuration
    • Show History - displays recently selected ontology terms in a list (under development)
  • Help
    • Phenote Help - brings up this help documentation within the Phenote Application. 
    • About - a splash screen about Phenote, including its current version number.

 

You can find out more information on each of these topics by looking through this documentation.

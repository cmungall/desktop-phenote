Menu Bars

Phenote has four menus with the following functions (many of which also have keyboard shortcuts):

  • File
    • Open - prompts the user to load a set of annotations saved to their computer. 
    • New - clears any currently loaded annotations and lets the user start over
    • Save/Save As - prompts the user to save the current annotations to a file
    • Exit - closes Phenote
  • Edit
    • Undo - undoes the last edit
    • Redo - redoes the last edit that you undid
    • Cut/Copy/Paste/Select All - these perform as in most applications. Note that before you can use these functions, you must click in the Annotation Table to bring the focus there.
    • Add Annotation - adds a new line to the Annotation Table
    • Duplicate Annotation - adds a copy of the currently selected annotation
    • Delete Annotation - deletes the currently selected annotation(s)
    • Enter and Select Next - this is just a reminder that if you hit return while you're in the Annotation Table, you will move to and select the next annotation after the one you're currently on.
    • Settings
      • The first four items under Settings are Search Filters - users can select the filters to be used during autocompletion of ontology fields.
        • Term Name
        • Synonyms
        • Definitions
        • Obsoletes
      • Set Configuration - allows the user to choose a different Phenote configuration or to set (or turn off) a proxy
      • Lock Docking Components - The components in the Phenote display can be moved around and detached as separate windows. This command locks them in place so you don't inadvertently move them. Select it again to turn off locking.
    • Help
      • User Guide - brings up this help documentation within the Phenote Application. 
      • About - brings up a small window showing the version number

     

    You can find out more information on many of these functions by looking through this documentation.

The <auto-update-ontologies> Tag

The <auto-update-ontologies> tag sets the default behavior for automatically updating ontologies when Phenote loads.

Parameters

Name Description Legal Values Required Default
enable If set to true, Phenote will automatically update an ontology file if a URL is given as the filename within the <ontology> tag
  • true
  • false
 no true

Contents

(none)

 

Example

<ns:auto-update-ontologies enable="true"/>

Running Phenote for the First Time

Choosing the Configuration

The very first time you start phenote, a window will popup and ask you to choose a configuration:

 

For demonstration purposes, we will work with the "demo" configuration.  Select this option and press OK.  Phenote will continue to load, but might take a few minutes to get started.  Since many ontolgies are updated everyday, it is important to keep in synch with the resources.  So, every time you start Phenote it checks to see if there are newer versions of ontologies available for download.  If so, it will automatically download the newest versions to your local computer.  (This feature can be disabled in the future.  See the Customizing Phenote section for more information.)

Phenote is both a complete piece of software and a software toolkit designed to facilitate the annotation of biological phenotypes using ontologies. It provides an interface to record genotype-phenotype pairs, together with the provenance for the annotation. Data annotated with Phenote is based on the EQ model for representing phenotypes, combining entities from any OBO ontology with qualities (such as PATO).

The software can be used as a complete package, or individual compontents can be downloaded and plugged in to your desired application. [Learn More]

This tool is being developed by the Bioinformatics Core of the National Center for Biomedical Ontology.

There are two groups driving the development of this software project:

  • Flybase: The Drosophila Genome Database.

  • ZFIN: The Zebrafish Genome Database

We are also collaborating with several groups:

NESCENT: The National Evolutionary Synthesis Center

PharmGKB: The Pharmacogenetics and Pharmacogenomics Knowledge Base

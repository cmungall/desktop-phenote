Launch Phenote in your browser

You can run phenote directly from your web browser without a separate installation by selecting one of the following links:

  • Stable release: 1.4
  • Older releases

You may be prompted to sign a certificate and allow the download. This will take a few minutes the first time you do this.

Run Phenote at the command line

To run phenote on the command line, linux or MacOSX users will need to checkout Phenote from the subversion repository. This can be done with the following command:

  • Stable release: svn co https://obo.svn.sourceforge.net/svnroot/obo/phenote/tags/1.4/ phenote
  • Latest: svn co https://obo.svn.sourceforge.net/svnroot/obo/phenote/trunk phenote

Once you've got it, you can run Phenote with ant with the command: ant run

Hardware requirements

  • 400 Mhz processor or faster.
  • 500+ MB RAM
  • 200 MB free disk space on your computer
  • Internet connection (broadband is best). Required for automatic updates of ontologies and some functions.

Software requirements

  • PC running Windows XP; Mac OSX; Linux
  • Webstart: Internet browser with webstart installed
  • Command line: Java 1.5 or later; Ant

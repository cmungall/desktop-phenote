Phenote+ installer edition

  • Platform-specific installers are available for Phenote+ v1.5 atsourceforge or here: MacOS X | Windows | Linux | Unix
    1. Download your platform-specific package. (For MacOS, this should self-extract into a disk image. If it doesn't, double-click on the file. You should see an icon for the "drive" that the Phenote+ application is on.
    2. Double-click on the Phenote installer icon. This will open up the installer software. You will be asked a few questions, such as the installation location.
    3. When its finished installing, you are ready to go! Double-click the Phenote application icon and its ready to run.

Check out our special documentation for running either on the command line or in Eclipse

Hardware requirements

  • 400 Mhz processor or faster.
  • 500+ MB RAM
  • 200 MB free disk space on your computer
  • Internet connection (broadband is best). Required for automatic updates of ontologies and some functions.

Software requirements

  • PC running Windows XP; Mac OSX; Linux; Solaris
  • Webstart: Internet browser with webstart installed
  • Command line: Java 1.5 or later; Ant
Recently added features, Phenote+ version 1.5
feature: New look and feel with Phenote+
  • Ability to move different panels around to customize the user experience. [1729114] These panels can be "locked" by going to the Settings->Lock Docking Components menu item. [1882110]
  • Panels can be "undocked" so that they can float independently around the screen
  • Phenote can interoperate with some obo-edit components, such as the tree-like dag viewer and the graph editor. [1822371]
feature: New ontology-update window shows the user in one place which ontologies need updating.
feature: if user changes their annotations since previous save, user is prompted for resave [1810448]
feature: "Save" added as menu item to overwrite currently open file [1803824]. A dot in the titlebar will disappear upon save. [1893249]
feature: Solaris OS now supported for webstart versions
feature: addition of the ProtocolEditor as a plug-in to phenote. For more information, please see the Protocol Editor demo. This plugin is being developed by the folks at the Open Microscopy Environment
feature: Term info panel updated with collapsible panes.
change: post-compositions are now displayed in multi-line style [1579357]
feature:Lookup abstracts from PubMed, if Pub field is configured.This can be tried in the "demo" configuration.
bugfix: Field name is now always shown, even if there is only one ontology configured for that field. It used to default to the ontology name in this instance. [1852112]
bufgix: Phenote can now accomodate empty E/Q fields when mapping to the annotation model [1894065]

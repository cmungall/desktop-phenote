Downloading Phenote

For users

Phenote installer edition

Platform-specific installers are available for Phenote v1.8.4 at SourceForge or here: MacOS X | Windows | Linux | Unix
  1. Download your platform-specific package. (For MacOS, this should self-extract into a disk image. If it doesn't, double-click on the file. You should see an icon for the "drive" that the Phenote application is on.)
  2. Double-click on the Phenote installer icon. This will open up the installer software. You will be asked a few questions, such as the installation location.
  3. When it's finished installing, you are ready to go! Double-click the Phenote application icon to launch Phenote.

We will continue to post links to major releases on this website. You can also install new releases from the Phenote SourceForge Project page.

Webstart edition

The webstart edition is currently unavailable while we resolve some problems. Please use the installer edition instead.

For developers

You can download the Phenote source code and compile and run it from the command line or in Eclipse.

Hardware requirements

  • 400 Mhz processor or faster
  • 500+ MB RAM
  • 200 MB free disk space on your computer
  • Internet connection (broadband is best). Required for automatic updates of ontologies and some functions.

Software requirements

  • Any of these operating systems: Windows XP; Mac OSX; Linux; Solaris
  • Java 1.6 or later
  • If you want to compile and run Phenote from the command line, you'll need Ant

Note about Phenote and Phenote+

In 2008, the Phenote developers introduced a new "look and feel" to the Phenote user interface. They called this new version "Phenote+". At that time, Phenote+ was considered a still-experimental version, and "regular" Phenote was still available. Starting with Phenote release 1.7, the developers decided that it's time to retire the old Phenote interface and make the Phenote+ interface the default. (Phenote+ will henceforth be referred to as "Phenote"; there won't be anything called "Phenote+".)
